e Pack for Visual Studio Code

=== Extensions ===
Atom Keymap (si usaste Atom, con esto podés usar los mismos atajos de teclado)

Color Highlight (Resalta el color HEX o RGB en tu CSS o HTML)

EJS Language Support (Soporte para archivo EJS - Templates de Express)

Beautify (Embellece el código HTML, CSS, y JS)

Emmet (ya viene instalado - shortcuts para teclado - https://docs.emmet.io/cheat-sheet/)
